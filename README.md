PageMinder - Library Management System

PageMinder is a simple web-based Library Management System built using
HTML, CSS, and JavaScript. It allows users to add, view, issue/return,
and delete books. Data is stored using the browserâ€™s localStorage, so no
backend is required.


Features:
- Add new books
- View book list
- Issue / Return books
- Delete books
- Preloaded default books
- Data persists after refresh


Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages
